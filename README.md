# Login System with Admin Support

A secure and responsive Login System built using HTML, CSS, JavaScript, and Local Storage concepts. This project includes user authentication along with Admin support for managing access and demonstrating role-based functionality.

## 🚀 Features

- User Login Authentication
- Admin Login Support
- Form Validation
- Role-Based Access
- Responsive User Interface
- Error Handling for Invalid Credentials
- Local Storage Integration
- Clean and Beginner-Friendly Code Structure

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## 📂 Project Structure

├── index.html
├── login.html
├── admin.html
├── style.css
├── script.js
└── assets/

## 🔐 Functionalities

### User Features
- User Login
- Credential Validation
- Access Control

### Admin Features
- Admin Authentication
- Separate Admin Access
- Basic Dashboard Functionality
